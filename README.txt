This repository replicates the analysis of the paper "When Is There Enough Data to Create a Global Statistic?" by Daniel Gerszon Mahler, Umar Serajuddin, and Hiroko Maeda. Thanks to the Stata module wbopendata, everything can be replicated without the need of auxilliary data files.

References:
Joao Pedro Azevedo, 2011. "WBOPENDATA: Stata module to access World Bank databases," Statistical Software Components S457234, Boston College Department of Economics, revised 02 Jul 2020.
